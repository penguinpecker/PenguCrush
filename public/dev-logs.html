To view logs from all users, connect to your Supabase project. Reading pengu_client_logs needs the service-role key (RLS blocks anon access).

⚠ The service-role key is powerful. It's stored only in this browser's localStorage and never committed or sent anywhere except your own Supabase project. Don't use this page on a shared/public machine, and don't deploy it publicly with a key entered.
